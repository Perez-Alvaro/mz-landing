     1  # MZ Landing Page
     2

styling.

mated with **Framer Motion**.

     7
     8  ```bash
     9  npm install

    11  ```
    12
    13  ## Building
    14
    15  ```bash

    17  npm run preview
    18  ```

    21
    22  - React
