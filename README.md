# MZ Landing Page

A responsive landing page for MZ featuring animated sections and modern styling.
Built with **React** and **Vite**, styled with **Tailwind CSS**, and animated with **Framer Motion**.

## Setup

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
npm run preview
```

## Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion
